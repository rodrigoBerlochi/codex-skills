# codex-skills

A small collection of skills adapted for the Codex AI agent.

These skills were sourced from different upstream repositories, then adjusted to fit Codex workflows and conventions. Attribution and upstream licensing notes are kept in each skill's `SKILL.md`.

This repository only includes a subset of skills that are useful for my current projects and day-to-day work. It is not intended to be a full catalog.

Free to use, adapt, and extend, subject to the license and attribution requirements noted for each imported skill.
